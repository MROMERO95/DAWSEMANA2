Sistema de Registro de Vehículos Desarrollo Web

Aplicación para administrar el ingreso y salida de vehículos en un parqueadero

Desarrollado por Ismenia Romero

Objetivo del sistema

Mejorar el control y organización del parqueadero mediante un registro digital sencillo.

Funciones principales

Permite registrar placas, hora de entrada y salida de cada vehículo.
