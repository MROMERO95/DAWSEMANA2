Inicio

Bienvenidos al sistema de registro de vehículos, una herramienta digital para controlar el ingreso y salida de vehículos en un parqueadero de forma eficiente y organizada.

Quiénes Somos

Este proyecto tiene como objetivo desarrollar un sistema web para el control de vehículos en un parqueadero. Permite registrar información como placa, hora de entrada y salida, mejorando la organización y reduciendo errores. Además, facilita la gestión del tiempo de permanencia de los vehículos y optimiza el uso del espacio disponible. Este sistema también tiene un propósito educativo, ya que permite aplicar conocimientos de HTML5, estructura semántica y diseño web básico. Su desarrollo ayuda a comprender cómo funciona una aplicación web real y fortalece habilidades en programación.

Servicios

  • Registro de vehículos
  • Control de entradas y salidas
  • Historial de parqueadero
